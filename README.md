# SmartHealthPredictor
<b>This is a streamlit web app that utilises Machine Learning models to predict certain diseases of human body!</b>

This app currently predicts 3 types of diseases :-
<ol>
<li>Diabetes</li>
<li>Heart Disease</li>
<li>Parkinsons</li>
